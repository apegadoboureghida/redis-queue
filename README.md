redis-queue
===============================

version number: 0.0.1
author: Andres Pegado

Overview
--------

A python package that can be installed with pip.

Installation / Usage
--------------------

To install use pip:

    $ pip install redis-queue


Or clone the repo:

    $ git clone https://github.com/apegadoboureghida/redis-queue.git
    $ python setup.py install
